## üìù Description

A Python module for **reading and visualizing** time-series data from **CSV log files** generated by **VCDS (VAG-COM Diagnostic System) OBD2** software, specifically from the **Measuring Blocks** function.

This repository includes a **Dash GUI** for interactive analysis and visualization of key parameters logged from the Vehicle Control Units (ECUs).

| Restriction | Description |

| **ECU Compatibility** | Only tested and guaranteed for CSV files generated with the **MARELLI 1AVP MPSF DG23** ECU. |
